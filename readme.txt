== Description ==

Allows embedding of WODTogether whiteboards into your blog posts automatically.

Learn more about [WODTogether](http://wodtogether.com/ "WODTogether.com")

== Installation ==

1. Upload `wodtogether-whiteboards.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure settings via the Settings / WODTogether page
